GeoAlert Public API

The GeoAlert Public API provides read-only access to aggregated, non-sensitive environmental monitoring data for public dashboards, third-party apps, and digital signage integrations.

Version v1
Protocol REST / JSON
Access Read-Only

Base URL

https://api.geoalert.city/v1

Authentication

Third-party developers can request an API key and include it in the request header. Public browser demos may use mock keys during development.

x-api-key: YOUR_API_KEY

Rate Limiting

The public API is rate-limited to protect availability and reduce scraping or abuse. Example limit: 60 requests per minute per API key.

429 Too Many Requests

Available Endpoints

All endpoints return JSON. These examples are mocked for the project demo.

GET /zones

Returns all supported monitoring zones available through the public API. 

[
  {
    "id": "downtownCore",
    "name": "Downtown Core"
  },
  {
    "id": "westIndustrial",
    "name": "West Industrial"
  },
  {
    "id": "northResidential",
    "name": "North Residential"
  }
]
GET /zones/:zoneId/current

Returns the latest aggregated environmental values for a single zone. 

{
  "zoneId": "downtownCore",
  "zoneName": "Downtown Core",
  "updatedAt": "2026-03-31T10:42:00Z",
  "metrics": {
    "airQualityIndex": 72,
    "temperatureC": 24,
    "humidityPercent": 58,
    "noiseDb": 81
  }
}
GET /zones/:zoneId/history?metric=airQuality&range=24h

Returns historical aggregated data points for one metric within a requested time range. 

{
  "zoneId": "downtownCore",
  "metric": "airQuality",
  "range": "24h",
  "points": [
    { "timestamp": "2026-03-31T06:00:00Z", "value": 61 },
    { "timestamp": "2026-03-31T08:00:00Z", "value": 67 },
    { "timestamp": "2026-03-31T10:00:00Z", "value": 72 }
  ]
}
GET /zones/:zoneId/aqi

Returns the current Air Quality Index summary for a zone for quick public display. 

{
  "zoneId": "downtownCore",
  "aqi": 72,
  "category": "Moderate",
  "updatedAt": "2026-03-31T10:42:00Z"
}
GET /alerts/public

Returns currently active public alerts that have already been acknowledged and published. 

[
  {
    "id": "alert-204",
    "zoneId": "westIndustrial",
    "type": "Air Quality",
    "severity": "High",
    "message": "Elevated particulate concentration detected.",
    "status": "Active"
  }
]
GET /health

Returns a lightweight status response indicating API availability. 

{
  "status": "ok",
  "service": "GeoAlert Public API",
  "version": "v1"
}

Example Request

GET /v1/zones/downtownCore/current HTTP/1.1
Host: api.geoalert.city
x-api-key: demo_public_key_123

Example Use Cases

  • Public environmental dashboards
  • Transit station or kiosk displays
  • Municipal open-data integrations
  • Student or research visualization tools
